About

This is the telegraph. The telegraph is a way for devices to communicate using ultrasonic frequencies of the sound domain(20154 Hz) to transmit data.

Currently, the telegraph can transmit binary data, so just a blink on other devices, but in the future the capabilities of the telegraph will expand.

I plan to add in the ability for the telegraph to transmit text data, encrypted text data, and also HTTP requests.

However this is the current working version of the telegraph so enjoy!

Transmission Log

This is the live transmission log of the telegraph. The box below will be red by defualt, but it will blink green when a binary 1 is sent over the wire.

Also, please note the input box. This box will have the number 200 in it by default, but this can be changed. The number controls what threshold determins if a signal is a one or if it is a zero. So if you want to bias the input data, a higher value will favor a zero and a lower value will favor a one. This number should be mapped between 0 and 255.
